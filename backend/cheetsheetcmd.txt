📝 Note: Laravel Migration Command
php artisan make:migration create_brands_table

Purpose:

Used in Laravel to create a migration file for a database table.

What it does:

Generates a new file in database/migrations/
The file is used to define the structure of the brands table

Why we use it:

Helps manage database structure using code
Acts like version control for the database
Makes it easy to share and update database schema

Next steps after running command:

Open the generated migration file
Define table columns (e.g., name, id, timestamps)

Run:

php artisan migrate

to create the table in the database
